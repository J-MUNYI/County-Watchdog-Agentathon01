# Problem and  Solution
County budgets are bulky documents and time consuming to go through each one of them. Watchdog helps the citizens by taking the lengthy documents and answering any questions asked via an agent, saving time and giving accurate information. Additionally it gives any updates to ammendments made to the documents.
# agent architecture
**Document AI & Vertex AI**: Ingests, pareses and indexes the county budget PDFs and complex financial tables.
**Gemini 1.5 Pro (Long COntext)**: Analyzes the dense extensive budget documents ensuring a deep contextual understanding across the hundreds of pages so as to answer citizen queries accurately.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
